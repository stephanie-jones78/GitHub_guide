## What This Covers

- **The mental model** — Git vs GitHub, why version control matters
- **Core concepts** — repo, commit, branch, staging, push/pull, merge
- **One-time setup** — SSH keys, Git config, everything you do once
- **The daily workflow** — the 7 commands you use every day
- **Branches** — how to build features without breaking what works
- **GitHub Pages** — deploy a live website free in under 5 minutes
- **Build in public** — habits that separate serious builders from hobbyists

## How to Use It

Open `index.html` in any browser. No build tools, no dependencies, no install. One file.

For full interactivity (tap cards, quizzes, copy buttons), open via a live URL rather than directly from your file system. See **Deploy** below.

## Deploy Your Own Copy

```bash
git clone https://github.com/yourusername/github-guide
cd github-guide
# Open index.html locally, or deploy via GitHub Pages
```

Enable GitHub Pages: **Settings → Pages → Deploy from main → Save**

Your live URL: `https://yourusername.github.io/github-guide`
